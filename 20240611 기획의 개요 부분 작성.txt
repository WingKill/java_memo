대표님의 기획 개요 짜기
-> 통계를 통한 자료 분석
-> 해당 부분을 통해 어떤 부분을 중점으로 둔 쇼핑몰 사이트를 만들 건지 정하는 것
(순서상)

- 개요 부분을 충당시킬 요소가 부족하다
	- 비교 기능이 필요하다. 	
		- 다른 쇼핑몰에 비교 가능한가 : '작업하는 쇼핑몰과 다른 쇼핑몰 간 가격 비교를 할 수 있는가?' 

- 해당 사이트가 상품성이 있는가를 따지는 중(2023 1인가구 소비 내역 통계를 기반으로 한 기획)
	- 시골 집 인테리어를 변경하여 전세로 내놓는 경우를 상정하면 인테리어 부분을 강조하는 게 좋지 않을까 싶음
	- 바닥, 마루, 타일, 벽지, 건축 자재를 하는 게 좋지 않을까...
	-> 서울이 아닌 다른 지역으로 알아보니 가능성이 보인다
	-> 대전, 세종에 사는 1인 가구들의 소비 지출 부분에서 가능성을 봄

- 커뮤니티 -> 비교 -> 상품선택 -> 비슷한 상품 간 가격을 비교 -> 연결 + 결제 서비스 제공

- 배송 서비스의 경우에도 요구사항을 추가하면 해당 요구사항에 따라(예를 들어 집 안에 들어와 놔달라고 하면 집 안에 놓고 가는 식)

- 동영상을 보는 비율도 높으므로, 동영상 시청 요소도 놓는 것을 고려하는 게 좋지 않을까 



-----> 인구 비율이 서울, 경기에 가장 많다는 부분을 확인하게 됨으로서 다시 짜게 됨

밸류 체인(value chain)
- 기업이 제품 또는 서비스를 기획하고 생산하여 고객에게 전달하기까지의 모든 활동과 과정에서 가치를 창출하는 각 단계를 의미

---

기획은 3가지

- 고객이 누군가?
- 그 고객들이 소비자로서의 역할을 하는가?
- 고객이 과연 의도한 방향으로 행동할 것인가?

-> 위에서는 젊은 층으로 타겟을 잡았지만, 베이비 붐 세대를 대상으로 짜는 게 제일 좋을 것 같다
