OSI 7계층

물리 계층 (Physical Layer): 네트워크 하드웨어와 관련된 실제 데이터 전송 처리. 케이블, 허브 등과 같은 장비들이 여기에 속함.

데이터 링크 계층 (Data Link Layer): 데이터 전송을 관리하고 에러 감지와 수정 수행. 이더넷과 같은 프로토콜들이 이 계층에 속함.

네트워크 계층 (Network Layer): 데이터를 패킷으로 나누어 다른 네트워크로 라우팅하고 목적지까지의 경로를 결정. IP 프로토콜이 여기에 속함.

전송 계층 (Transport Layer): 데이터 전송을 관리하고 신뢰성 있는 전송을 보장. TCP와 UDP가 이 계층에서 작동.

세션 계층 (Session Layer): 통신 세션을 설정, 유지, 종료하며 동기화를 제공.

표현 계층 (Presentation Layer): 데이터의 형식을 변환하고 암호화, 압축 등의 기능을 수행하여 응용 계층으로 전달.

응용 계층 (Application Layer): 최종 사용자의 응용 프로그램과 직접 상호 작용. HTTP, FTP, SMTP 등의 프로토콜들이 이 계층에서 동작.
